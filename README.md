## Objectif :
Création d'un blog
* 2 types de model :
    * Articles
        * Titre
        * Description
        * Contenu
        * Auteur
    * Utilisateurs
        * Nom
        * Prenom
        * email
* Il faut un CRUD pour
    * Articles
    * Utilisateurs
* Une page d'authentification qui va simuler le login d'un utilisateur.
